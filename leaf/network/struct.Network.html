Defines a Network that contains the Layers and Blobs that store the intermediate results between the layers which are generated by forward/backward.

It is also responsible for setting up the connections between the layers. A Network is usually used together with a Solver to optimize the networks' weights.

Computes forward step for a network whose input blob references have been set and returns the output blobs.

Can be used if you need more control over how to put data into the network (debugging), otherwise forward is the prefered method to forward through the whole network.

fn forward_from_to(&mut self, start: usize, end: usize) -> f32

Compute forward step for a part of (or the whole) network and returns the total loss.

Computes the forward step from the layer with index start to the layer with index end and return the total scalar loss over all loss layers.

If you want to compute a foward step for the whole network you should use forward_prefilled. Computing a forward on a part of the network is usually only done for debugging purposes.

fn backward(&mut self)

Computes a backpropagation step for the whole network using the currently set output blobs.

Computes the backpropagation step for each layer of the Network using backward_from_to.

Called directly only for debugging purposes. Backpropagating a network is only useful during training and handled by a Solver
